# Imágenes de la App AMVA Móvil

## Logo Principal

Coloca aquí el archivo del logo de AMVA Móvil.

**Nombre del archivo:** `logo-amva-movil.png` (o `.jpg`, `.webp`)

**Descripción del logo:**
- Globo terrestre con continentes en verde y océanos en azul
- Texto "A.M.V.A" en color dorado/naranja sobre el globo
- Texto "movil" en azul a la derecha del globo
- Fondo negro

**Formatos recomendados:**
- PNG con transparencia (preferido)
- JPG de alta calidad
- WebP para mejor compresión

**Tamaños recomendados:**
- Logo principal: 512x512px o mayor
- Versión para splash: 1024x1024px o mayor
